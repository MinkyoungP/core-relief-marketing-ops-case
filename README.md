# CoreRelief Marketing Ops / Growth Case Study

## Project Overview
This project is a Marketing Operations and Growth case study based on a Direct-to-Consumer (DTC) brand, CoreRelief.

The objective of this project is to analyze contribution margin by customer segment and evaluate whether reallocating marketing budget can improve profitability while maintaining revenue growth.

This project focuses on business decision-making, budget allocation strategy, and scenario simulation from a Marketing Operations perspective.

---

## Business Problem
CoreRelief wants to understand whether reallocating marketing budget across customer segments can improve contribution margin without significantly reducing total revenue.

Some segments generate higher revenue but lower margins, while others generate lower revenue but higher margins.  
This creates a trade-off between revenue growth and profitability.

The key question:
> Can budget reallocation improve overall contribution margin, or will rising CPA offset the expected gains?

---

## Analysis Scope
This project includes the following analyses:

- Business Model Overview
- Contribution Margin Analysis by Segment
- Budget Reallocation Simulation
- Scenario Analysis with CPA Changes
- Assumptions and Limitations Documentation

---

## Key Insights
- High-revenue segments were not always the most profitable segments.
- Budget reallocation improved the contribution margin structure.
- However, increased CPA reduced the expected revenue improvement.
- Budget strategy should consider both revenue and acquisition efficiency.
